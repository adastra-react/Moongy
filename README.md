## Moongy TVShows

#### Overview:

Moongy TVShows 

An application used to fetch and displaty searched movies using the TVMaze API.

## Installation and Setup Instructions

#### Example:  

Clone this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`yarn install`  

To Run Test Suite:  

`yarn test`  

To Start Server:

`yarn start`  

To Visit App:

`localhost:3000`
